## 2. `twd_transfer` — 法幣（台幣）加值提領

| 欄位名稱 | 型別 | 說明 | 範例值 |
|---|---|---|---|
| `created_at` | `datetime` | 法幣出入金訂單建立時間 | `2025-10-09 11:57:33` |
| `user_id` | `bigint unsigned` | 發起此法幣出入金的使用者 ID | `26` |
| `kind` | `tinyint` | 交易類別：`0` 加值（入金）、`1` 提領（出金） | `1` |
| `ori_samount` | `decimal(65,0)` | 實際總金額（原始值）；**真實台幣金額 = 欄位值 × 1e-8** | `998500000000` |
| `source_ip` | `bigint` | 下單時使用者 IP（整數格式）；**注意：外部匯入加值時此欄位為空值** | `3544954843` |

## 3. `crypto_transfer` — 虛擬貨幣加值提領

| 欄位名稱 | 型別 | 說明 | 範例值 |
|---|---|---|---|
| `created_at` | `datetime` | 虛擬貨幣交易建立時間 | `2025-10-12 07:02:52` |
| `user_id` | `bigint unsigned` | 發起此交易的使用者 ID | `10023` |
| `kind` | `tinyint` | 交易類別：`0` 加值（入金）、`1` 提領（出金） | `1` |
| `sub_kind` | `tinyint` | 子類別：`0` 外部（鏈上交易）、`1` 內部（交易所內轉） | `0` |
| `ori_samount` | `decimal(65,0)` | 虛擬貨幣實際原始數量；**真實數量 = 欄位值 × 1e-8** | `814764500` |
| `twd_srate` | `decimal(65,0)` | 交易當下幣種對台幣匯率；**真實匯率 = 欄位值 × 1e-8** | `2047830000` |
| `relation_user_id` | `bigint unsigned` | 內轉發送/接收對方的 user_id | `96357` |
| `from_wallet` | `varchar(191)` | 發送方的區塊鏈來源錢包地址 | `addr1vx7j284...` |
| `to_wallet` | `varchar(191)` | 接收方的區塊鏈接收地址 | `addr1vx7j284...` |
| `protocol` | `tinyint` | 區塊鏈協定/主網（見下方 ENUM 對照） | `4` |

| `source_ip` | `bigint` | 下單時使用者 IP（整數格式）；**注意：從外部錢包加值時此欄位為空值** | `313891166` |

## 4. `usdt_twd_trading` — 掛單簿 USDT/TWD 交易對成交訂單

| 欄位名稱 | 型別 | 說明 | 範例值 |
|---|---|---|---|
| `updated_at` | `datetime` | 訂單完成時間 | `2025-10-02 23:23:25` |
| `user_id` | `bigint unsigned` | 下單的使用者 ID | `2713` |
| `is_buy` | `tinyint(1)` | 是否為買單：`0` 賣單（USDT → TWD）、`1` 買單（TWD → USDT） | `1` |
| `trade_samount` | `decimal(65,0)` | 已成交數量（Base）；**真實數量 = 欄位值 × 1e-8** | `262103105` |
| `twd_srate` | `decimal(65,0)` | 實際成交的對台幣匯率；**真實價格 = 欄位值 × 1e-8** | `3054000000` |
| `source_ip` | `bigint` | 下單時使用者 IP（整數格式，站內交易皆會記錄） | `27356118` |
| `is_market` | `tinyint(1)` | 是否為市價單：`0` 限價單、`1` 市價單 | `0` |
| `source` | `int` | 訂單來源終端：`0` WEB、`1` APP、`2` API | `1` |

## 6. 金額換算說明

> ⚠️ 所有涉及金額、數量、匯率的欄位皆以整數形式儲存，使用時需進行換算。

| 欄位類型 | 換算方式 | 範例 |
|---|---|---|
| 台幣金額（`ori_samount`） | `× 1e-8` | `998500000000 × 1e-8 = 9985 TWD` |
| 虛擬貨幣數量（`ori_samount`, `trade_samount` 等） | `× 1e-8` | `814764500 × 1e-8 = 8.147645` |
| 匯率（`twd_srate`） | `× 1e-8` | `3054000000 × 1e-8 = 30.54 TWD/USDT` |
